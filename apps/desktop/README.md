# Yap Desktop (Tauri)

One small cross-platform app (Windows/Mac/Linux) that replaces the helper
scripts. Joins as a normal peer and injects received text **at the OS cursor** —
including on Linux/Wayland via the XDG RemoteDesktop portal.

**Build instructions:** [`/prompts/desktop.md`](../../prompts/desktop.md)
